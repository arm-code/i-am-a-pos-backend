# Proyecto backend para la gestión de puntos de venta


#### Descripción
Este proyecto es la `API` que tiene como proposito la gestión de diversos puntos de venta con diversos enfoques. La idea es que pueda funcionar con diferentes enfoques; desde un punto de venta común y otros como renta o prestación de servicios. 

---
#### Configuración del proyecto para su ejecución
- Clonar el repositorio en nuestra maquina.
- Utilizar forzosamente `pnpm` para la gestión de las librerias de Javascript.
- Ejecutar `pnpm install`

#### Variables de entorno de para conectar al servidor de base de datos
- Crear un archivo `.env` (en la raiz del proyecto) basado en `.env.template`
- Colocar las credenciales correspondientes.

#### Correr el proyecto en localhost
- Ejecutar `pnpm run start:dev`
